## Tense-Aspect
- Conjugates.

|            |                 | **Tenses** | **Tenses**  | **Tenses** |
| ---------- | --------------- | ---------- | ----------- | ---------- |
|            |                 | **Past**   | **Present** | **Future** |
| **Aspect** | **Cont.**       | -kōva-     | -kōra-      | -kōca-     |
| **Aspect** | **Perf.**       | -pāva-     | -pāra-      | -pāca-     |
| **Aspect** | **Stative**     | -tēva-     | -tēra-      | -tēca-     |
| **Aspect** | **Simple**      | X          | -naira-     | -naica-    |
| **Aspect** | **Perf. Cont.** | -ḷima-     | -ḷima-      | -ḷica-     |
stem + tense/asp + person ending
