# Vāstuguku Pinnaganyāḍi
Yēṣva gu pinnaganyāḍikidailu caluppuke kamṭa Vāstuguku. Yēganyāḍili, yēṣvaku mīnḍa sāgganyāḍipi caltērasācalpu: 
- Vāstuguku Pinnaganyāḍi
- Taṣṭinupuku Pinnaganyāḍi

Vāmstegī, Taṣṭinupuku Pinnaganyāḍili, Sōmyamu Mīsāḍinuku gu pinnakēyutaṣṭiganyāḍi chaltērasācalpu.



"This is a treatise about our mother Earth. In this book, there are two sub-treatises:
- The 'Vāstuguku Pinnaganyāḍi' (the encyclopaedia of the world)
- The 'Taṣṭinupuku Pinnaganyāḍi' (the encyclopaedia of languages)

Predominantly, in the 'Taṣṭinupuku Pinnaganyāḍi', there is a dictionary of Classical Mishtu."
